StringBuffer和StringBuilder

  • StringBuffer/StringBuilder可以看做可变长度字符串。
  • StringBuffer/StringBuilder初始化容量16。
  • StringBuffer/StringBuilder是完成字符串拼接操作的，方法名：append()。
  • StringBuffer是线程安全的。StringBuilder是非线程安全的。
  • 频繁进行字符串拼接不建议使用“+”,因为这样会产生很多对象。

包装类

  • 包装类，就是将基本数据类型用一个类进行一次包装，使得其可以按照引用数据类型就行访问。
  • 基本数据类型和包装类型对照表
基本数据类型 包装类
byte Byte
short Short
int Integer
long Long
float Float
double Double
boolean Boolean
char Character

    • 由基本数据类型向包装类型转换。
  • 拆箱：
    • 由包装类型向基本数据类型转换。
